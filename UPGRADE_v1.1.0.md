# 升级到 v1.1.0 指南

## 🎉 欢迎升级到 v1.1.0！

本版本带来了更好的用户体验和新功能。

## 🆕 主要改进

### 1️⃣ 自动加载活动
- 打开网页即可看到所有活动
- 无需任何额外操作
- 加载状态清晰可见

### 2️⃣ 单字段对比
- 对比更直观，每个活动一条曲线
- 支持快速切换不同指标
- 性能更优，渲染更快

## 📥 升级步骤

### 方式一：全新安装（推荐）

1. **下载新版本**
   ```
   fitanalysis_v1.1.0.zip (38.6 MB)
   ```

2. **备份数据**（如果有旧版本）
   - 复制旧版本的 `data/` 文件夹

3. **解压新版本**
   - 解压到任意目录

4. **恢复数据**
   - 将备份的 `data/` 文件夹复制到新版本目录

5. **启动应用**
   - Windows: 双击 `run.bat` 或 `fitanalysis.exe`
   - Linux/Mac: 运行 `./run.sh`

6. **访问应用**
   - 浏览器打开 `http://localhost:8080`

### 方式二：原地升级

1. **备份数据**
   ```powershell
   Copy-Item data data_backup -Recurse
   ```

2. **下载并解压v1.1.0到临时目录**

3. **替换文件**（保留data目录）
   - 替换 `frontend/` 文件夹
   - 替换 `backend/` 文件夹  
   - 替换 `run.bat` 和其他脚本

4. **重启应用**

## ⚠️ 注意事项

1. **备份数据**：升级前务必备份 `data/` 目录
2. **关闭旧版本**：确保旧版本应用已关闭
3. **端口占用**：确保8080端口未被占用
4. **浏览器缓存**：升级后建议清除浏览器缓存（Ctrl+Shift+Delete）

## 🐛 常见问题

### Q: 升级后看不到活动列表？
**A**: 检查 `data/` 目录是否正确复制，刷新浏览器（Ctrl+F5）

### Q: 页面显示"加载中..."但一直不显示？
**A**: 
1. 检查后端服务是否正常启动
2. 查看浏览器控制台是否有错误
3. 确认 `data/index.json` 文件存在

### Q: 对比功能无法使用？
**A**: 清除浏览器缓存，强制刷新页面（Ctrl+Shift+F5）

### Q: 数据丢失了？
**A**: 从备份的 `data/` 目录恢复
