# FIT跑步数据分析器 v1.2.0 发布说明

**发布日期**: 2025-12-09  
**版本号**: v1.2.0

## 📦 发布说明

本次发布为正式打包发布版本，优化了发布包结构，移除了所有测试代码和测试数据，提供更轻量级的生产环境部署包。

## 🔧 优化改进

### 1. 发布包优化
- 🗑️ **移除测试代码**: 排除所有test_*.py测试文件
- 🗑️ **移除诊断工具**: 排除diagnose_fit.py和verify_vosc_fix.py
- 🗑️ **移除测试数据**: 不包含示例FIT文件
- 📦 **减小体积**: 发布包更加轻量化

### 2. 构建配置改进
- ⚙️ **优化spec配置**: 更新PyInstaller排除列表
- 📝 **版本信息更新**: 统一版本号到v1.2.0
- 🔧 **构建流程优化**: 仅打包必需文件

## 📋 发布包内容

**核心文件**:
- `fitanalysis.exe` - 主程序可执行文件
- `启动服务器.bat` - Windows启动脚本
- `_internal/` - 依赖库和运行时文件
  - `backend/` - 后端核心代码（不含测试文件）
  - `frontend/` - 前端界面文件
- `data/` - 数据存储目录（空目录，首次运行自动创建）
- `README.md` - 用户使用说明
- `RELEASE_v1.2.0.md` - 本发布说明

**排除的文件** (开发环境专用):
- 测试文件: test_api.py, test_iq_fix.py, charts.test.js
- 诊断工具: diagnose_fit.py, verify_vosc_fix.py
- 构建文件: build.py, fitanalysis.spec
- 开发文档: agent.md, BUGS.md
- 测试数据: 546218476_ACTIVITY.fit
- 历史发布文档: RELEASE_v1.1.0.md, RELEASE_CHECKLIST_v1.1.0.md

## ✅ 功能特性

### 核心功能 (继承自v1.1.0)
- ✅ FIT文件上传和解析（支持所有标准字段和IQ扩展字段）
- ✅ 活动列表自动加载和管理
- ✅ 单活动详细数据展示
- ✅ 多活动单字段对比
- ✅ 秒级趋势图（可选字段，X轴时间/距离切换）
- ✅ CSV数据导出
- ✅ 本地JSON存储（无需数据库）

### 技术特性
- 🌐 **Web界面**: 基于FastAPI + 原生JavaScript
- 📊 **图表渲染**: Plotly.js交互式图表
- 💾 **数据存储**: JSON文件存储，无需外部数据库
- 🔒 **本地运行**: 127.0.0.1:8080，数据不离开本地

## 📥 安装使用

### 首次安装

1. **下载**: 下载 `fitanalysis_v1.2.0.zip`
2. **解压**: 解压到任意目录（如 `C:\Apps\fitanalysis\`）
3. **启动**: 双击运行 `启动服务器.bat`
4. **访问**: 浏览器打开 http://127.0.0.1:8080
5. **使用**: 上传FIT文件开始分析

### 从旧版本升级

1. **备份数据**: 复制旧版本的 `data/` 目录
2. **安装新版**: 按首次安装步骤操作
3. **恢复数据**: 将备份的 `data/` 目录替换到新版本目录
4. **验证**: 启动后确认活动列表正常显示
